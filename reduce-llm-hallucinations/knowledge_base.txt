LangSmith 是一个用于开发、调试和部署大语言模型应用的平台。
它可以帮助开发者追踪每一次 LLM 调用，查看 Prompt、输出、Token 消耗和错误信息。

LangGraph 是一个用来编排有状态 LLM 工作流的框架。
它非常适合构建多步骤、多 Agent、可循环的推理流程，并且可以和 LangSmith 无缝集成进行可观测性监控。

DashScope（通义千问）是阿里云提供的大模型平台，包含通义千问系列 LLM 和文本向量嵌入模型等能力。
通过 LangChain 的 ChatTongyi 和 DashScopeEmbeddings 即可方便地集成到 Python 应用中。
